Conclusión

Dependiendo de las características del proyecto (a mi parecer es ideal para pequeños proyectos que no necesitan un Back End para desplegar Base de Dato), Netlify es una muy buena opción, si se quiere desplegar algo gratuito, como por ejemplo Landing Pages, Portafolios o pequeñas prácticas. Además tiene la opción de brindar una URL ".netlify" o configurarla a una propia, por lo cual me parece muy buena opción para Landing Pages de pequeños emprendimientos.

Por otro lado GitHub Pages también es gratuita, pero la suelo ver mas usada en pequeñas prácticas de programadores, para desplegar Portafolios o proyectos personales. Tiene la gran ventaja de ya estar integrada en GitHub, por lo que el despliegue prácticamente no requeiere de mucha configuración.

Si bien tiene sus limitaciones, la opción gratuita de AlwaysData es la adecuada para uun proyecto BackEnd que necesite crear una API y manejar su Base de Datos.
